# ubnt_wifi_exporter
Metric exporter for ubnt wireless controllers to use with Prometheus

# Remarks
This is an early version. It uses undocumented API calls so it can break at any  time.

# Install
```
go get github.com/czerwonk/ubnt_wifi_exporter
```
